## 📦 التثبيت والتشغيل

### 1. متطلبات النظام

- Node.js (الإصدار 14 أو أحدث)
- MongoDB (الإصدار 4.4 أو أحدث)
- npm أو yarn

### 2. Backend Setup

```bash
cd hospital-management-system/backend

# تثبيت المكتبات
npm install

# إنشاء ملف .env (انظر docs/environment-variables.md)
cp docs/environment-variables.md .env
# قم بتعديل الملف حسب إعداداتك

# إنشاء المستخدم الأدمن الأولي
npm run setup

# أو إنشاء المستخدم الأدمن + مستخدمين تجريبيين
npm run setup:full

# ملء قاعدة البيانات ببيانات تجريبية شاملة (175+ سجل)
npm run seed

# تشغيل الخادم
npm run dev
```

### 3. Frontend Setup

```bash
cd hospital-management-system/frontend
npm install
npm start
```

### 4. تسجيل الدخول الأولي

بعد إنشاء المستخدم الأدمن، يمكنك تسجيل الدخول بالبيانات التالية:

- **البريد الإلكتروني:** admin@maherhospital.com
- **اسم المستخدم:** admin
- **كلمة المرور:** admin123

🚨 **مهم:** قم بتغيير كلمة المرور فور تسجيل الدخول الأول!

## 🔐 نظام المصادقة والأدوار

### الأدوار المتاحة

1. **super_admin** - مدير النظام الرئيسي

   - جميع الصلاحيات
   - إدارة المستخدمين والأدوار

2. **admin** - مدير المستشفى

   - إدارة المرضى والأطباء والمواعيد
   - عرض التقارير والإحصائيات

3. **doctor** - طبيب

   - عرض وتحديث بيانات المرضى المحولين إليه
   - إدارة السجلات الطبية والمواعيد

4. **nurse** - ممرض/ممرضة

   - عرض وتحديث بيانات المرضى
   - إنشاء السجلات الطبية

5. **receptionist** - موظف استقبال

   - إدارة المرضى والمواعيد
   - إنشاء الفواتير

6. **pharmacist** - صيدلي

   - عرض بيانات المرضى والوصفات الطبية

7. **lab_technician** - فني مختبر

   - إدارة نتائج الفحوصات المختبرية

8. **radiologist** - أخصائي أشعة

   - إدارة نتائج الأشعة

9. **accountant** - محاسب

   - إدارة الفواتير والتقارير المالية

10. **manager** - مدير قسم
    - عرض التقارير والإحصائيات

### مسارات المصادقة

```
POST /api/auth/login          - تسجيل الدخول
POST /api/auth/logout         - تسجيل الخروج
GET  /api/auth/me             - بيانات المستخدم الحالي
POST /api/auth/change-password - تغيير كلمة المرور
GET  /api/auth/verify-token   - التحقق من صحة التوكن
```

### مسارات إدارة المستخدمين

```
POST /api/users               - إنشاء مستخدم جديد
GET  /api/users               - جميع المستخدمين
GET  /api/users/:id           - مستخدم محدد
PUT  /api/users/:id           - تحديث بيانات المستخدم
DELETE /api/users/:id         - حذف المستخدم
```

## 🌐 المسارات الرئيسية (API Endpoints)

### المرضى

- `GET /api/patients` - جميع المرضى
- `POST /api/patients` - مريض جديد
- `GET /api/patients/:id` - مريض محدد
- `PUT /api/patients/:id` - تعديل مريض
- `DELETE /api/patients/:id` - حذف مريض

### الأطباء

- `GET /api/doctors` - جميع الأطباء
- `POST /api/doctors` - طبيب جديد
- `GET /api/doctors/:id` - طبيب محدد
- `PUT /api/doctors/:id` - تعديل طبيب

### المواعيد

- `GET /api/appointments` - جميع المواعيد
- `POST /api/appointments` - موعد جديد
- `GET /api/appointments/:id` - موعد محدد
- `PUT /api/appointments/:id` - تعديل موعد
