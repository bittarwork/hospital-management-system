## 🚀 التشغيل السريع

### المتطلبات الأساسية

- Node.js (v16 أو أحدث)
- MongoDB (v5 أو أحدث)
- npm أو yarn

### 1. تحميل المشروع

```bash
git clone [repository-url]
cd hospital-management-system
```

### 2. إعداد Backend

```bash
cd backend
npm install

# إنشاء ملف البيئة
cp .env.example .env

# تشغيل السيرفر
npm start
```

### 3. إعداد Frontend

```bash
cd frontend
npm install

# تشغيل واجهة المستخدم
npm run dev
```

### 4. إنشاء البيانات التجريبية

```bash
# في مجلد backend
node scripts/createAdminUser.js
node scripts/seedPatients.js
```

## 🛠️ التقنيات المستخدمة

### Backend

- **Node.js** - بيئة تشغيل JavaScript
- **Express.js** - إطار عمل الخادم
- **MongoDB** - قاعدة البيانات
- **Mongoose** - ODM للـ MongoDB
- **JWT** - أمان المصادقة
- **bcryptjs** - تشفير كلمات المرور

### Frontend

- **React.js** - مكتبة واجهة المستخدم
- **Vite** - أداة البناء السريعة
- **TailwindCSS** - إطار عمل التصميم
- **Framer Motion** - الحركات والانتقالات
- **React Router** - التنقل
- **Axios** - طلبات HTTP
- **React Hot Toast** - الإشعارات
