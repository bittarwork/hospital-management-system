# 🏥 نظام إدارة مستشفى ماهر العلي - مشروع التخرج

## 📦 التثبيت والتشغيل

### 1. تحميل المشروع

```bash
git clone <repository-url>
cd hospital-management-system
```

### 2. تثبيت Backend

```bash
cd backend
npm install
```

### 3. تثبيت Frontend (مُحسّن مع Vite ⚡)

```bash
cd ../frontend
npm install --legacy-peer-deps
```

### 4. إعداد قاعدة البيانات

```bash
# تأكد من تشغيل MongoDB أولاً
mongod

# أو استخدم MongoDB Atlas (السحابة)
# قم بتحديث MONGODB_URI في ملف الإعدادات
```

### 5. تشغيل النظام

#### Backend (المنفذ 5000)

```bash
cd backend
npm run dev
```

#### Frontend (المنفذ 3000) - Vite ⚡

```bash
cd frontend
npm run dev
```

### 6. الوصول للنظام

- **Frontend:** http://localhost:3000
- **Backend API:** http://localhost:5000/api
- **Health Check:** http://localhost:5000/health

## 🌐 المسارات الرئيسية (API Documentation)

### مسارات المرضى

```
GET    /api/patients           # جميع المرضى
POST   /api/patients           # إضافة مريض جديد
GET    /api/patients/:id       # مريض محدد
PUT    /api/patients/:id       # تعديل مريض
DELETE /api/patients/:id       # حذف مريض
GET    /api/patients/stats     # إحصائيات المرضى
```

### مسارات الأطباء

```
GET    /api/doctors            # جميع الأطباء
POST   /api/doctors            # إضافة طبيب جديد
GET    /api/doctors/:id        # طبيب محدد
PUT    /api/doctors/:id        # تعديل طبيب
GET    /api/doctors/:id/schedule # جدول الطبيب
```

### مسارات المواعيد

```
GET    /api/appointments       # جميع المواعيد
POST   /api/appointments       # حجز موعد جديد
GET    /api/appointments/:id   # موعد محدد
PUT    /api/appointments/:id   # تعديل موعد
GET    /api/appointments/today # مواعيد اليوم
```

### مسارات السجلات الطبية

```
GET    /api/medical-records    # جميع السجلات
POST   /api/medical-records    # سجل جديد
GET    /api/medical-records/:id # سجل محدد
PUT    /api/medical-records/:id # تعديل سجل
```

### مسارات الفواتير

```
GET    /api/invoices           # جميع الفواتير
POST   /api/invoices           # فاتورة جديدة
GET    /api/invoices/:id       # فاتورة محددة
PUT    /api/invoices/:id       # تعديل فاتورة
GET    /api/invoices/stats     # إحصائيات مالية
```
